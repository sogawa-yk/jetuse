# specs/01 — デザインシステム（Redwood風）

状態: **ドラフト**。SPIKE-07 のギャラリーを人間がレビューして確定する。

## 結論（plan.md §1 より）

React + Tailwind + Redwood風テーマ。Oracle JET は不採用（理由はADRで明文化、SPIKE-07で最終確認）。

## デザイントークン

- **プライマリ**: Oracle Red 系（`#C74634` 近傍）。アクセントは控えめ。
- **背景**: Redwood特有の暖色ニュートラル（クリーム〜グレージュ）。ダークモードあり。
- **形状**: フラット + 細罫線基調。角丸・余白は控えめなRedwoodトーン。AWSコンソール風の青系・高密度UIにしない。
- **フォント**: Oracle Sans は使わない（OSS配布考慮）。Inter + Noto Sans JP を採用し `theme` で差し替え可能に。

## リブランド要件

ロゴ・プライマリカラー・プロダクト名は `branding.json` 1ファイルで差し替え可能にする。デフォルトが「Oracleっぽい」というだけの状態を保つ。
