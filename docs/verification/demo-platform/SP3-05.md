- 大阪 vector store: 空 store 作成は回復(36s completed)したが **file 索引化は依然失敗**(2026-07-08
  13時時点)。プローブは「store 作成」だけでは不十分 — file attach まで見るべき。

  リージョンローカル。大阪 OCID を渡すと 400 "Invalid OpenAI project")。jetuse-dev には
  `jetuse-loop-project`(ORD)が既存 — 再利用できる。
- 生成 runtime はスキャフォールドの **vendored node_modules が前提**(オフラインビルド)。fresh
  worktree では `GENERATION_SCAFFOLD_DIR` で実体を指すか npm ci が必要(配備イメージ化は後続タスク)。
