# OPS-01 検証レポート: 管理ダッシュボード

- 日付: 2026-06-13 / ブランチ: `task/ops-01` / 仕様: specs/15-hardening.md

## 実装

- `/admin` ページ（ナビ「管理」、SPAのみ。APIは SEC-02 の `/api/admin/usage` を流用）
- 表示: サマリ3枚（総リクエスト/総トークン/概算コスト）+ 日別トークン消費バー +
  機能別/モデル別/ユーザー別(上位50)テーブル + 概算コスト（モデル別の粗い単価）
- 集計期間 7/30/90日 切替
- 認可: サーバー側 `ADMIN_USERS` で制御。非管理者は403 → 画面に「管理者のみ」を表示
- データ源は監査ログ(ADB)。OCI Monitoring(OPS-02)はアラーム/長期傾向用に併用する位置づけ

## 検証

- ローカルでシードデータ55件を投入し描画確認（`img/OPS-01-admin.png`）:
  サマリ・日別バー・3テーブル・概算コストがすべて表示
- 本番API(0.26.0)は `/api/admin/usage` 実装済み・管理者(daybreaks.link@gmail.com)設定済み →
  SPAデプロイのみで利用可能
