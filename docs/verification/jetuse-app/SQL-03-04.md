## SQL-03: 結果の自動グラフ化

- `POST /api/dbchat/chart`: llama（高速）に列+サンプル行を渡しチャート仕様（type/x/y/title/reason）をJSON提案させ、**列名の実在検証**をしてから返す（不適は type=none）
- UI: 結果テーブルの「📊グラフ化」→ Chart.js（動的import・チャンク分離）で描画。種別の手動上書き可
- 実機: チャネル別売上に `bar`/X=CHANNEL_DESC/日本語タイトルが提案され描画OK

## SQL-04: Select AI直接実行モード

- `Nl2SqlRequest.backend: sql_search | select_ai`。Select AIプロファイル（JETUSE_SQL_AI、llama、object_list=SH、comments=true）をJETUSE_APPスキーマに遅延作成
- UI: 生成ボタン横のバックエンドセレクタ
- 実機: 初回8秒（プロファイル作成込み）で正答SQL生成・実行
