## 実機で確定したハマりどころ（tipsにも記録）

1. **CrewAI 1.14はlitellm非依存**（openai SDK直）。IAM署名は `LLM(interceptor=...)` の
   `BaseInterceptor.on_outbound` で注入できる。**`provider="openai"` の明示が必須**
   （`openai/モデル名` 形式はモデル名がOpenAI既知リストと照合され、不一致だとlitellm
   フォールバック→未導入エラーになる）
2. CrewAIの依存chromadbが **sqlite3≥3.35** を要求（OL9標準は古い）→ `pysqlite3-binary` を
   入れて `sys.modules["sqlite3"]` を差し替える定石で回避
3. **AI SDK（TS）は custom fetch + `oci-common` の `DefaultRequestSigner`** で署名注入できる
   （`createOpenAI({fetch})`）。Node 18+の `fetch` には `duplex: 'half'` が必要
4. 全FW共通: `CompartmentId` + `OpenAi-Project` ヘッダ必須（ADR-0007の誤誘導エラーに注意）
