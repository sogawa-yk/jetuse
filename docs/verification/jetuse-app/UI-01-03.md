## UI-01: Redwoodトークン抽出

- `@oracle/oraclejet` を作業用ディレクトリ（`~/tmp-oraclejet`、リポジトリ外）にインストールし、
  `packages/web/scripts/extract-redwood-tokens.mjs` で `dist/css/redwood/oj-redwood.css`（約1.05MB、全文読み込みなし）から抽出。
- 結果: **ライト 1,066変数 / ダーク（再定義） 374変数**。`var()` 多段参照の未解決 0件・循環 0件。
- 確認ポイント: `--oj-palette-neutral-rgb-170` = `49, 45, 42`（**#312D2A**）をライトで抽出 → ヘッダー色として採用。
- システムフォントスタック: `--oj-html-font-family` = `-apple-system, BlinkMacSystemFont, "Segoe UI", "Helvetica Neue", Arial, sans-serif` を抽出し採用（Oracle Sans不使用。index.htmlのGoogle Fonts読み込みも廃止）。
- ソースCSS側の `rgba(var(色var), .6)` 入れ子1件が `rgba(#hex, .6)` という不正値になるため、`rgba(r, g, b, .6)` へ正規化する救済処理を実装。
- 出力: `packages/web/src/styles/tokens.css`（:root=ライト / `[data-theme="dark"]`=ダーク）、`docs/ui/tokens-report.md`（2026-06-12 人間レビュー承認済み）。

## UI-02: 実装

コンポーネント分解（人間レビュー承認済み）どおりに実装:

| 部品 | 実装先 |
|---|---|
| AppHeader（#312D2Aヘッダ・中央ピル型検索バー・リージョン表示・アバター） | `src/components/layout.tsx` Shell |
| SideNav(白面・選択は `--oj-core-bg-color-selected`) / AppFooter(ダーク) | 同上 |

- 色のハードコードは全廃。意味トークン（`--rw-` / `--oci-` 系）→ `tokens.css` のRedwood実値を参照する2層構成
  （Tailwind v4 `@theme inline` のユーティリティ名は維持し、既存ページの変更を最小化）。
- 例外: コンソール緑（チャット吹き出し・チップの `--oci-band` 系4色）はRedwoodに該当トークンがなく、
  スクリーンショット優先ルールに基づき `theme.css` に集約定義。
- ダークモード: `.dark` クラスに加え `data-theme="dark"` を `<html>` に付与（`prefs.tsx`）し、
  `tokens.css` のダーク変数と連動。
- ヘッダ検索バーは実機能（ホームのユースケース/エージェント絞り込み `/#/?q=...`）。

## UI-03: Playwright自己検証（ループ1周で収束、上限2周）

- 環境: Playwright MCP + Google Chrome（OL9は `npx playwright install chrome` 非対応のためRPM直接インストール）。
  ヘッドレスの日本語/絵文字描画に `google-noto-sans-cjk-ttc-fonts` / `google-noto-emoji-color-fonts` が必要。
- devサーバーは `VITE_AUTH_REQUIRED=false` で起動（`.env.local` は変更しない）。ルートはHashRouterのため `/#/path`。

| 周回 | 発見したズレ | 対処 |
